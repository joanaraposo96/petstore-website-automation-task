# Petstore Task Automation

This project is a task for the **Petstore website**, demonstrating automated interactions using **Playwright**. The main functionalities covered are:

## Features

- **Search a product** – find products by name or category  
- **View detailed product information** – access product descriptions, prices, and details  
- **Add a product to the cart** – simulate adding items to the shopping cart  

## Tech Stack

- **Framework:** Playwright  
- **Language:** JavaScript
