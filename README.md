# Petstore Task Automation

This project is a task for the **Petstore website**, demonstrating automated interactions using **Playwright**. The main functionalities covered are:

## Features
- **Login and Logout** - access the user account and verify the logged-in status on the homepage.
- **Search a product** – find products by name or category  
- **View detailed product information** – access product descriptions, prices, and details  
- **Add a product to the cart** – simulate adding items to the shopping cart  

## Tech Stack

- **Framework:** Playwright  
- **Language:** JavaScript

## Setup & Usage

Follow these steps to get the project running on your machine:

1. **Clone the repository**  
```bash
git clone https://github.com/joanaraposo96/petstore-search-add-to-cart.git
cd petstore-search-view-add-to-cart
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the Playwright test**
```bash
npx playwright test --ui
```
